# TypeScript Hello World

Minimal TypeScript Node project.

## Setup

Windows PowerShell:

```powershell
cd typescript-hello-world
npm install
```

macOS / Linux:

```bash
cd typescript-hello-world
npm install
```

## Scripts

- `npm run dev` — Run in development with automatic restart (requires `ts-node-dev`).
- `npm run build` — Compile TypeScript to `dist/`.
- `npm start` — Run the built output from `dist/`.
- `npm run clean` — Remove `dist/`.

## Example

Development:

```bash
npm run dev
```

Build and run:

```bash
npm run build
npm start
```
